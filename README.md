# Throuple Tea Studio OS — Smart Update

This update keeps OBS and the podcast recorder as the real recording systems. Studio OS is the production intelligence and control layer around them.

## Apple-inspired design update

- Native dark-mode hierarchy and typography
- Floating glass navigation
- Quieter translucent surfaces and compact controls
- Mobile bottom navigation dock
- Apple TV-style shared teleprompter treatment
- Throuple Tea neon color used as an accent rather than covering every surface
- Reduced-motion support

## Smart production layer

The new Studio Intelligence panel reads the actual shared Studio state and surfaces the most useful next action.

It can identify:

- Incomplete preflight checks
- When the studio is ready to begin
- A segment running over its target
- An empty question board during a planned discussion segment
- Must-mentions still open near the end
- A long recording with no edit markers
- Uneven manually tracked host airtime
- A paused timer requiring attention
- Postflight steps still incomplete
- Sensitive / Do Not Publish markers

## Manual airtime tracking

Speaker share is based on the William, Daniel and Caleb speaker buttons. It does not listen to or analyze audio. The timer now records the duration between speaker changes and displays a live estimated share.

## Production confidence

The percentage combines:

- Preflight completion
- Must-mention completion
- Current segment timing
- Marker coverage during the recording

It is a workflow indicator, not an audio-quality score.
